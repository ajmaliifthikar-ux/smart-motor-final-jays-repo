## Phase 2 — Ultimate Site Architecture & URL Taxonomy

### **Master Slug Strategy (Exact Naming)**

| Page Type          | Ideal Slug                                      | SEO Rationale                                                              |
| ------------------ | ----------------------------------------------- | -------------------------------------------------------------------------- |
| Service Category   | `/services/paint-protection-film-ppf-abu-dhabi` | Includes service, abbreviation, and geo modifier for strong local matching |
| Brand + Repair     | `/brands/mercedes-benz-repair-abu-dhabi`        | Uses canonical brand name + service type + geo modifier                    |
| General Brand Info | `/brands/mercedes-benz`                         | Pure brand value page for informational + top-of-funnel                    |
| Blog / Tips        | `/blog/ceramic-coating-abudhabi-benefits`       | Clear long-tail pattern                                                    |
| Booking            | `/book-service` (with query params)             | Clean conversion endpoint                                                  |

**Key Notes**

* Avoid broad generic paths like `/services/detailing` — instead include **what + where** (e.g., “PPF Abu Dhabi”).
* Stick to **kebab-case**, no spaces or underscores — consistent crawlable taxonomy.

---

### **Hierarchy & Cannibalization Blueprint**

```
/                      → Homepage  
/services              → Service landing hub (overview + featured)  
/services/[service]    → Individual service (PPF, ceramic, diagnostics)  
/brands                → All brands overview  
/brands/[brand]        → Brand value page (informational)  
/brands/[brand]/services  → Brand+Service cluster (repair, maintenance)  
/blog                  → Smart Tips content hub  
/blog/[post]           → Individual article (SEO optimized)  
/book-service          → Primary booking funnel
```

**Avoid cannibalization between `/services` and `/brands`**

* `/services/ceramic-coating` should target *transactional intent*.
* `/brands/mercedes-benz` targets *informational + brand interest*.
* Brand pages should **link down to services** via contextual CTAs (e.g., “Mercedes ceramic coating — check pricing & warranty”).
